# Heartopia MIDI Player Launcher

This launcher is designed to make it easier to open and play MIDI files using [Heartopia-Midi-Player](https://github.com/DonElf/Heartopia-Midi-Player/releases/tag/v1).

## Features
- Quickly open `.mid` files with Heartopia-Midi-Player (works like **Open With**).
- Add multiple MIDI files to choose from.
- Provides a button to directly close Heartopia-Midi-Player to stop playback.

## Installation
1. Download **Heartopia-Midi-Player** from [Release v1](https://github.com/DonElf/Heartopia-Midi-Player/releases/tag/v1).
2. Make sure you have the file `HeartopiaMidiPlayer.exe` on your system.

## Usage
1. Launch the program.
2. Use **Browse** to select the `HeartopiaMidiPlayer.exe` file.
3. Add the MIDI files you want to open.
4. Press **Play** to open a MIDI file with Heartopia-Midi-Player  
   (each time you open a file, playback starts from the beginning).
5. Press **Stop** to close the `HeartopiaMidiPlayer.exe` program.

## Notes
- This launcher only serves as a helper to run Heartopia-Midi-Player.
- Heartopia-Midi-Player must be installed before the launcher can be used.
